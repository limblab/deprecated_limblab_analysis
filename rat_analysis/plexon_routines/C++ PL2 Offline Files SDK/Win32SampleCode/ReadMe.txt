Plexon C/C++ SDK for Reading PL2 Files

The SDK contains:
- Binaries for 32-bit and 64-bit builds of PL2FileReader.dll and a test program PL2FileReaderTest.exe
- Header files describing functions available in PL2FileReader.dll
- Visual Studio 2005 console project demonstrating how to use various functions in PL2FileReader.dll


Please see PL2FileStructures.h for definitions of various structures used in the PL2 file access functions.

Please see PL2FileReader.h for documentation on all the functions available in PL2FileReader.dll

Please see PL2FileReaderTest.cpp for sample code that uses PL2FileReader.dll
